# Contact YDB-Qdrant

Use the public GitHub repository for project questions, bug reports, API behavior reports, package usage, documentation gaps, and reproducible self-hosting problems. Use the Code Indexer support page for hosted GitHub App, dashboard, repository indexing, and MCP-token issues.

## Best channel by topic

- GitHub issues: https://github.com/astandrik/ydb-qdrant/issues
- Code Indexer support: https://ydb-qdrant.tech/code-indexer/support/
- Developer resources: https://ydb-qdrant.tech/developers/
- REST API docs: https://ydb-qdrant.tech/docs/api/
- Auth docs: https://ydb-qdrant.tech/docs/auth/

## What to include

Include the endpoint or page URL, expected behavior, observed response, request ID if an API error returned one, and whether the issue affects self-hosted REST, the public demo endpoint, or hosted Code Indexer MCP. Do not share production API keys, MCP tokens, GitHub OAuth tokens, private repository contents, or customer data in public issues.

## Agent guidance

AI agents should read `https://ydb-qdrant.tech/openapi.json`, `https://ydb-qdrant.tech/llms-full.txt`, and `https://ydb-qdrant.tech/.well-known/agent.json` before recommending integration steps. The root vector REST product and the hosted Code Indexer MCP product have separate auth models and separate scopes.
