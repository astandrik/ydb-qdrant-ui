# YDB-Qdrant pricing

YDB-Qdrant is available as an Apache-2.0 open-source package and a self-hosted HTTP server.

## Package and self-hosting

- Price: USD 0 for the `ydb-qdrant` package and self-hosted YDB-Qdrant server.
- License: Apache-2.0.
- Users pay their own YDB, compute, storage, network, backup, and monitoring costs.
- No hosted production SLA is published for the root YDB-Qdrant REST product.

## Code Indexer public beta

- Price: USD 0 while public beta access is available.
- The hosted beta limits repository count, indexed chunks, and daily searches.
- MCP tokens are created and revoked in the dashboard.
- No enterprise, SLA, or paid hosted plan is published today.
